# Deteksi Kelelahan - Fatigue Detection System

Sistem deteksi kelelahan multi-modal berbasis computer vision menggunakan Streamlit dan OpenCV.

## Fitur
- PERCLOS (Eye Closure Ratio)
- Yawn Detection (MAR)
- Head Pose Estimation
- Multi-modal Fusion

## Deploy
Aplikasi ini dideploy di Streamlit Cloud.
